# Unicode fonts for PDF (Telugu & Hindi)

To show **Telugu** and **Hindi** text correctly in the homework PDF (instead of English fallback), place these font files here:

- **NotoSansTelugu-Regular.ttf** – for Telugu subject
- **NotoSansDevanagari-Regular.ttf** – for Hindi subject

**Download:**
- Noto Sans Telugu: https://fonts.google.com/noto/specimen/Noto+Sans+Telugu (download ZIP, use the Regular .ttf)
- Noto Sans Devanagari: https://fonts.google.com/noto/specimen/Noto+Sans+Devanagari (download ZIP, use the Regular .ttf)

Or from GitHub (google/fonts):
- https://github.com/google/fonts/tree/main/ofl/notosanstelugu
- https://github.com/google/fonts/tree/main/ofl/notosansdevanagari

If these files are missing, the PDF will show Telugu/Hindi homework in **English** so text is never gibberish.
